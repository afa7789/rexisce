rexisce

Native XMPP desktop messenger written in Rust using iced.

Built with vibecoding, taking reference from open-source XMPP clients like Halloy, Dino, and Gajim.

Download

Pre-built binaries are attached to each GitHub release.

PlatformFile
Linux (x86_64)rexisce-linux-x86_64
macOS (Intel)rexisce-macos-x86_64
macOS (Apple Silicon)rexisce-macos-aarch64
Windowsrexisce-windows-x86_64.exe

Build from source

Requires Rust 1.80+.

git clone https://github.com/owner/rexisce
cd rexisce
make build        # release binary in target/release/
make run          # debug build + run
